# タイトル

CustomButtonYoutube

# 概要

Youtube画面上にあるアイコンの、表示 / 非表示を切り替えるChrome拡張機能です。<br>
キャスト機能を間違えて押してしまい、家族用のTVへ映してしまうという悲劇を繰り返さないよう作成しました。<br>
また、Chromeだと常に隠れているPIP機能(常時デスクトップ右下に表示する機能)を表示したり、アイコンは普段使うもののみに絞ってスッキリさせておきたい方にもおすすめです。
